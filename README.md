# Simple Command Line Calculator

A straightforward, easy-to-use command-line calculator written in C. This program allows users to perform basic arithmetic operations (+, -, *, /) on two numbers directly from the terminal. 

## Features

* **Basic Arithmetic:** Supports Addition (`+`), Subtraction (`-`), Multiplication (`*`), and Division (`/`).
* **Interactive Prompts:** Guides the user with clear console instructions and formatted output lines.
* **Error Handling:** Prevents program crashes by explicitly checking for and blocking division by zero.
* **Precision Output:** Results are formatted to two decimal places for readability.

## Prerequisites

To compile and run this program, you will need a C compiler installed on your system. 
* **Windows:** [MinGW](https://www.mingw-w64.org/) or [Visual Studio Build Tools](https://visualstudio.microsoft.com/downloads/)
* **macOS:** Install Xcode Command Line Tools by running `xcode-select --install` in the terminal.
* **Linux:** Install GCC (usually included by default). If not, run `sudo apt install build-essential`.
